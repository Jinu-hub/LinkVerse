# LinkVerse
An expansive memory store that organizes and connects all personal experiences and knowledge.

## Overview
단순히 링크를 저장하는 도구가 아닙니다. 
개인의 관심, 경험, 지식을 유연하게 정리하고
기억하고 회고할 수 있도록 돕는
**개인화된 브레인 플랫폼**입니다.
